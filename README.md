
# University Management System


## Overview
The University Management System is a CLI-based Python application designed to streamline university academic and administrative operations. A modular approach is used to provide specialized functions and interfaces for different roles such as students, lecturers, administrators, accountants and registrar. Data such as student records, attendance and module enrollments are managed using text files acting in place of a proper database.
## Features

- CRUD Operations for managing students, modules or lecturers
- Handling student registration through generation of unique student IDs
- Allows students to check their grades and view attendance records
- Enables acadmemic staff to manage student grades
- Manage financial records and transactions

## Skills

- Data Management using .txt files
- Modular Programming
